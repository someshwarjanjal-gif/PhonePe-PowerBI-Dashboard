# 📊 PhonePe Transaction Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard built to analyze PhonePe transaction data and provide business insights.

## Tools Used

- Power BI
- Microsoft Excel
- DAX

## Features

- Total Transactions
- Total Transaction Value
- Successful Payment Rate
- Monthly Transaction Trend
- Age-wise Customer Analysis
- Top 5 Users
- Weekday vs Weekend Analysis
- Service Transaction Analysis

- 300K+ Transactions

- Millennials contribute the highest share
- Weekday transactions are higher than weekend transactions
